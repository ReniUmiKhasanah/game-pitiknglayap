🎮 Deskripsi Game (Kelompok)

Nama Game: Pitik Nglayap: Cross the Road

Genre: 2D Endless Runner / Traffic Dodger

Pitik Nglayap adalah sebuah game arcade berbasis web bertema kearifan lokal yang mengisahkan seekor ayam kampung nekat yang kabur dari kandang demi menjelajahi dunia luar. Kali ini, si Pitik harus melompati jalan raya antar kota yang sangat padat dan ramai.

Pemain bertanggung jawab penuh untuk memandu si Pitik menyebrang dengan selamat, menghindari marabahaya lalu lintas berupa kendaraan bermotor (seperti mobil sedan, truk logistik, hingga sepeda motor) yang melaju kencang. Jika salah memperhitungkan waktu lompatan, si Pitik akan tertabrak kendaraan dan memicu kondisi Game Over.

Fitur Utama Game:
  • Tema Jalan Raya: Arena berlatar belakang aspal hitam lengkap dengan marka jalan putih putus-putus dinamis dan trotoar rumput pedesaan.
  • Varian Kendaraan Acak: Rintangan didesain bervariasi secara prosedural mulai dari mobil, truk, hingga motor dengan warna yang berbeda-beda.
  • Peningkatan Tantangan Progresif: Intensitas lalu lintas kendaraan bertambah padat dan melaju lebih kencang seiring bertambahnya skor pemain.

Peran dalam Projek: Game Logic & Sprite Generator Programmer

Kontribusi mandiri saya pada projek ini berfokus pada transisi tema mekanika rintangan dari objek statis (batu) menjadi objek dinamis (kendaraan berlalu-lintas) menggunakan teknik rendering grafik komputer langsung pada komponen HTML5 Canvas. Saya merancang arsitektur object generator menggunakan struktur data JavaScript array untuk menampung properti koordinat, dimensi panjang kendaraan, serta roda objek yang bergerak dinamis.

Saya juga menerapkan perhitungan deteksi benturan berbasis koordinat persegi (*Axis-Aligned Bounding Box*) yang presisi untuk mendeteksi kapan tepatnya karakter Pitik berbenturan dengan bemper kendaraan depan maupun belakang. Seluruh manajemen kode program ini di-maintain dan di-hosting secara publik memanfaatkan ekosistem GitHub Pages.
